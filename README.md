# LlanquihueTourApp - Semana 6

## Objetivo

Implementar una jerarquía de clases usando herencia simple en Java,
aplicando reutilización de atributos, constructores con super() y
sobreescritura del método toString().

## Clases creadas

### ServicioTuristico
Clase base con:
- nombre
- duracionHoras

### RutaGastronomica
Hereda de ServicioTuristico.
Incluye:
- numeroDeParadas

### PaseoLacustre
Hereda de ServicioTuristico.
Incluye:
- tipoEmbarcacion

### ExcursionCultural
Hereda de ServicioTuristico.
Incluye:
- lugarHistorico


## Ejecución

Ejecutar la clase:

ui/Main.java

El programa mostrará los servicios turísticos creados por consola.
