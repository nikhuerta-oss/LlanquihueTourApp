# LlanquihueTourApp

Sistema de gestión para la agencia turística Llanquihue Tour.

## Paquetes utilizados

model:
Contiene las clases Persona, Guia y Operador.

service:
Contiene la lógica de gestión de datos.

util:
Contiene la lectura del archivo CSV.

app:
Contiene la clase principal Main.

## Ejecución

Ejecutar la clase Main.java para iniciar el sistema.
